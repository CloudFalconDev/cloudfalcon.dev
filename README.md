## Tech Stack

- **Next.js 16** - App Router
- **React 19** - UI
- **Tailwind CSS v4** - Styling
- **Framer Motion** - Animations
- **Three.js** - 3D Background
- **Biome** - Linting & Formatting

## Development

```bash
# Install
npm install

# Run dev server
npm run dev

# Lint & format
npm run check

# Build
npm run build

```

Open [localhost:3000](http://localhost:3000)
